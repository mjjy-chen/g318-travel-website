# G318 Travel Website - 318进藏自驾游

A comprehensive travel guide website for the Sichuan-Tibet Highway (G318), one of the most scenic and challenging routes in China.

## Features

- 🏔️ **Scenic Spots Display**: Detailed information about major attractions along the route including Yala Snow Mountain, Ranwu Lake, Basumtso Lake, 72 Hairpin Turns, and Midui Glacier
- 🏨 **Accommodation Listings**: Recommended places to stay at key stops with ratings, price ranges, and special features
- 🍜 **Food and Dining Guide**: Local specialties and must-try dishes from Tibetan and Sichuan cuisine
- 🎭 **Local Customs and Culture**: Information about Tibetan Buddhist traditions, festivals, and cultural etiquette
- 📱 **Fully Responsive**: Works on mobile devices, tablets, and desktop computers
- 🎨 **Travel-Themed Design**: Color palette inspired by the landscape (mountain purple, lake cyan, culture orange, road emerald)

## Implementation Details

This website uses:
- HTML5 semantic structure
- CSS3 with custom properties for easy theming
- Vanilla JavaScript (no frameworks)
- Responsive design principles

## Content Sections

1. **Hero Section**: Introduction to the G318 journey
2. **Route Highlights**: Key attractions categorized by type (mountains, lakes, culture, road wonders)
3. **Travel Essentials**: Best time to visit, vehicle preparation, altitude sickness prevention, communication tips
4. **Photo Gallery**: Interactive sections for scenery, accommodation, food, and culture
5. **Call to Action**: Invitation to get the complete travel guide
